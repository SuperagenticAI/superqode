# Enterprise

Enterprise adds powerful automation, deep evaluation testing, and integrations on top of the OSS core.

## OSS vs Enterprise

| Capability | OSS | Enterprise |
| --- | --- | --- |
| CLI automation | Basic | Powerful + deep evaluation testing |
| Deep QE (agentic scans) | ✅ | ✅ |
| Fix suggestions + verification | ❌ | ✅ |
| Test generation & scenario packs | ❌ | ✅ |
| CI outputs (JSONL/JUnit) | ❌ | ✅ |
| QE dashboard & artifacts UI | ❌ | ✅ |
| Expert prompt packs | ❌ | ✅ |
| Optimized SuperQE mode | ❌ | ✅ |
| IDE integrations (VSCode/LSP) | ❌ | ✅ |
| Natural Language QE | ❌ | ✅ |
| ML severity model | ❌ | ✅ |

| Moltbot integration | ❌ | ✅ |

## Included Capabilities

- Fix suggestions + verification
- Test generation and scenario packs
- CI outputs (JSONL/JUnit)
- QE dashboard and artifacts UI
- IDE/LSP and server integrations
- Natural language QE
- Prompt packs and optimized workflows
- ML severity model
- Enterprise integrations (Moltbot first; more bot integrations coming)
- Moltbot integration ([https://www.molt.bot/](https://www.molt.bot/))

## Enterprise Integrations

Enterprise integrations let you connect SuperQode/SuperQE with external agent platforms in a
self-hosted setup. Moltbot is the first supported integration; more bot integrations will be added
over time.

Contact the Superagentic AI team at [info@super-agentic.ai](mailto:info@super-agentic.ai) for access.
